Project - Version Management

Manage the versions of your projects

Handle your releases and versions: Create a new version for your project that groups all the tasks that will be in the next release.

This module comes with 5 version states:

  1. Draft
  2. Work In Progress
  3. Staging
  4. Production
  5. Past

User guide

How to use this module

  1. Go to Search > Versions and create a Version
  2. If you want to add a new Version State, go to Configuration > Version States
  3. Add tasks to the desired Version
  4. Advance the Version State when needed

Notes

  • The Version States which have Done State checked can only be activated if all their tasks are in the stage Done
  • When a version of a Project is set to the Production state, the previous Production version of this project is automatically set to the state Past
