User guide

How to use this module?

  1. Create a workflow by using the create form in Configuration > Workflows.
  2. In Configuration > Task Steps create as many steps as needed and assign them to the workflow.
  3. Create a task type in Configuration > Task Type linking it to your workflow.
  4. A new field named type now appears in your tasks.
  5. Set it to your task type and you will see the task steps of the appropriate workflow appear at the top right corner of the task.
  6. To move the task in the different steps, use either the next step button or just click on the desired step.
